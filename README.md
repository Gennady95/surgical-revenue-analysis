# Подсчёт хирургической части в выручке отделения пластической хирургии
Скрипт анализирует хирургическую часть выручки, исключая сопутствующие услуги (анестезию, палату, доп. продажи), формирует детальную аналитику по дням, месяцам, кварталам и годам, а также рассчитывает структуру выручки и топ услуг по каждому хирургу. Используется для расчёта премий, оценки эффективности работы хирургов и анализа структуры продаж.

# Surgical Revenue Analysis

> Python application for calculating the surgical portion of revenue and analyzing surgeon performance.

## Description

This project analyzes completed surgical procedures and separates the revenue directly generated by surgeons from related services such as anesthesia, inpatient care and additional sales.

The application produces detailed analytical reports that allow management to evaluate surgeon performance, calculate bonus payments and analyze the structure of surgical revenue over different time periods.

Important:

The project requires access to the hospital database.

Without the original database structure the application cannot be executed.

## Business Goal

The application was developed to restore and automate the calculation of surgeon bonus payments.

By separating the surgical component of revenue from supporting services, management can accurately evaluate each surgeon's financial performance and analyze the contribution of specific procedures.

## How It Works

1. Loads completed surgical procedures
2. Filters only surgical revenue items
3. Excludes related non-surgical services
4. Calculates revenue for each surgeon
5. Aggregates statistics by day, month, quarter and year
6. Identifies top-performing procedures
7. Builds detailed Excel reports
